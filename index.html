Features

🎯

Multi-camera fusion

Supports up to 6 cameras simultaneously for wide-area coverage. Auto-detects and validates feeds.

🧠

Real-time 3D tracking

MediaPipe HandLandmarker with One-Euro filtering and 6-frame moving averages for jitter-free control.

🎚️

Stereo calibration

Full stereo calibration with a printed checkerboard. Produces a shared world frame for all cameras.

♿

Accessibility-first

Focus navigation via swipes + Tab/Shift+Tab. Fullscreen overlay shows current focus. No cursor needed.

🔒

100% local

No cloud, no telemetry, no API keys. Everything runs on your machine. Optional Ollama integration.

📦

One-command install

Cross-platform installer (Windows, Linux, macOS). Installs Python, dependencies, and model in one go.
