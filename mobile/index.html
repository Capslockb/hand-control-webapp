Quick answers

Does it cost anything?
No. MIT-licensed open source. Free forever, no trial, no upsell.
Does it work without internet?
Yes. The hand-tracking pipeline is 100% local. The first launch downloads a 7 MB model file; after that no network is used (unless you explicitly enable the optional Ollama gestures).
What cameras work?
Any UVC webcam — built-in laptop cam, USB webcam, or virtual camera. For 3D room mapping, two or more cameras spaced 10–30 cm apart.
Does it drain my battery?
At idle (no hands in frame), it uses ~3% of one CPU core and ~200 MB RAM. On a laptop this is roughly the same as a browser tab.
Can I uninstall it?
Yes. Delete the folder. The installer keeps everything self-contained — no system files touched, no registry entries on Windows, no daemons on Linux/macOS.

Cross-platform install + test fixes. Now actually works on Linux, macOS, and headless systems.

Test status: 70/71 pass on Linux (1 environmental fail is the real-webcam test).
